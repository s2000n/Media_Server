# Local Media Server 🎬

منصة متكاملة لإدارة وعرض الوسائط الرقمية (مقاطع الفيديو والصور) تعمل بالكامل داخل شبكتك المحلية (**Local Network**). تم تصميم هذا المشروع ليكون بديلاً محليًا آمنًا، سريعًا، ومستقلاً تمامًا عن خدمات التخزين السحابي (**Cloud Storage**)، مما يمنح المستخدم التحكم الكامل في بياناته وخصوصيته.

---

## ✨ الميزات الرئيسية (Features)

### 📁 تنظيم وإدارة ذكية للوسائط
* **مرونة الفرز:** إمكانية إنشاء مجلدات مخصصة وفرز مقاطع الفيديو والصور حسب نوع الميديا، وتقسيمها بما يتناسب مع طريقة العرض المفضلة.
* **إدارة متكاملة من الواجهة:** واجهة مستخدم سلسة تتيح نقل المواد المرئية بين المجلدات المختلفة، أو إعادة تسميتها، أو حذفها مباشرة من المتصفح دون الحاجة للتعامل مع ملفات النظام يدويًا.

### 📊 لوحة تحكم تفاعلية (Smart Dashboard)
* **مراقبة التخزين الفورية:** واجهة رسومية واضحة تعرض مساحة التخزين المستخدمة والمتبقية على الأقراص لحظة بلحظة.
* **التحليل البياني:** دعم الرسوم البيانية التفاعلية (Graphs) تحليل حجم وتوزيع المحتوى ونسبته بدقة داخل كل مجلد لفهم استهلاك المساحة.

### 🔒 نظام متقدم لإدارة المستخدمين والأمن الداخلي
* **توزيع وتحجيم الصلاحيات:** إمكانية إنشاء حسابات مستقلة (مثل حسابات أفراد العائلة مخصصة للمشاهدة فقط، مقابل حسابات مشرفين بصلاحيات كاملة).
* **إدارة الرتب (Role Management):** مرونة عالية في ترقية الحسابات إلى مشرف (Admin) أو خفضها إلى مستخدم عادي (User) بضغطة زر.
* **الحماية والتشفير:** إمكانية إعادة تعيين كلمات السر وتعديل بيانات الحسابات، مع **تشفير كامل لكافة كلمات المرور داخل قاعدة البيانات** لضمان أعلى مستويات الحماية.
* **سجل تسجيل الدخول (Session Tracking):** تتبع حالة الحسابات ومعرفة تاريخ ووقت آخر عملية تسجيل دخول ناجحة لكل مستخدم.

### ⚡ تخزين محلي معزول (Privacy & Performance)
* **أداء فائق وبدون تقطيع:** عزل كامل للبيانات عن الإنترنت الخارجي لحماية الخصوصية. تصفح وتشغيل مقاطع الـ 4K أو الصور ذات الدقة العالية يتم بسرعة فائقة دون أي انتظار (Zero Buffering)، حيث تعتمد السرعة كليًا على كفاءة شبكتك الداخلية.

> 📝 **تنويه:** تم رفع قاعدة بيانات جاهزة للاستخدام مباشرة؛ يمكن استيرادها عن طريق MySQL Server وتحتوي على مستخدم Admin افتراضي يمكن تسجيل الدخول إليه مباشرة ببيانات التدقيق التالية: `admin:admin`.
